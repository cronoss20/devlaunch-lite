# Contributing

1. Create a feature branch.
2. Install development dependencies with `pip install -e '.[dev]'`.
3. Run `ruff check .`, `mypy src` and `pytest`.
4. Open a pull request describing the problem and the chosen solution.

Use Conventional Commits where practical, for example:

- `feat: add interactive project wizard`
- `fix: protect existing output directories`
- `test: cover invalid project names`
